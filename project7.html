Traffic lights controller

In this project a simple traffic lights controller is designed with STM32F401 microcontroller.
Software language: C.
RTOS: FreeRTOS.

MATERIEL
STM32F401 Discovery
4 digit seven segment display
2 x RED/GREEN/YELLOW LED
2 x BC547 Transistor
Push button
13 x 220 ohm Resistors
3 x 1K ohm Resistors
