ORIENTATION TRACKING WITH ADXL345 SENSOR

The DE10-Lite FPGA board contains a 3-axis (X, Y and Z) ADXL345 accelerometer chip (commonly referred to as the "G-Sensor") which may be used to measure the orientation of the board. This sensor uses the I2C protocol for communication with the Nios II processor. The output values from the sensor actually depend on the selected sensitivity, which can vary from +- 2g to +-16g, the default sensitivity is +-2g.
